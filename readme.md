# 辩论赛计时器

使用HTML+CSS+原生JS制作而成，可以运行在任意现代浏览器。

## 使用说明

不建议使用

但你要用的话：左键下一页，右键上一页，按滚轮启停计时器，滚滚轮调音乐音量

## 核心原理

在slides变量中存储页面列表，每一项代表一个页面，
每一项有3个函数，分别在左键、右键、滚轮键按下时调用。

## 亮点

将计时器定义为一个类，方便使用

界面炫酷。很有B格（bushi

## 改进建议

将各种计时界面封装为不同的类，使用更方便

将页面切换写成一个函数，避免反反复复写`.animate(...)`

将`document.getElementByID`重命名为`$`，避免反反复复敲

优化切页逻辑，**尤其是连续点会导致排版炸掉的问题！**
